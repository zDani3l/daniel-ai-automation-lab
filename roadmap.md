# Roadmap

## Phase 1: Foundations

- Learn Python syntax by building small tools
- Understand functions, dictionaries, lists, files, and errors
- Use GitHub as a public learning record
- Publish weekly progress on LinkedIn

## Phase 2: Backend And APIs

- Build simple APIs with FastAPI
- Understand routes, request bodies, responses, and validation
- Connect APIs to useful workflows
- Document every project with examples

## Phase 3: Automation

- Build scripts that save time
- Connect APIs and external services
- Automate repeated business tasks
- Learn how to identify useful automation opportunities

## Phase 4: AI Workflows

- Learn LLM basics
- Build AI assistants for specific workflows
- Use prompts, structured outputs, and APIs
- Build small tools that combine backend logic and AI

## Phase 5: Portfolio And Opportunities

- Publish polished GitHub repositories
- Improve LinkedIn positioning
- Build case studies
- Look for freelance, collaboration, and remote opportunities
